# Poligorn website

Минималистичный RU/EN сайт‑портфолио разработчика игр (Poligorn) с витриной проектов (Itch.io — **ручной список**), резюме и блоком студии **Tea Agency Games**.

## Локальный запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
npm run preview
```

Сборка складывается в `dist/` — это удобно для деплоя на статический хостинг (Reg.ru).

## Контент

- Соцсети: `src/content/socials.ts` (сейчас заглушки — нужно проставить реальные ссылки)
- Проекты: `src/content/projects.ts` (источник правды — ручной список, типизировано, RU/EN поля)
